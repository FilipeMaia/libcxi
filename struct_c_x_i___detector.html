Detailed Description

Holds information about one of the detectors used during the experiment. Raw data recorded by a detector as well as its position and geometry should be stored in this class.

Member Data Documentation

double CXI_Detector::basis_vectors[2][3]

A matrix with the basis vectors of the detector data. For more details see the CXI File Format reference 8.2.1.

int CXI_Detector::basis_vectors_valid

Is 1 if basis_vectors is set and 0 if not.

double CXI_Detector::corner_position[3]

The x, y and z coordinates of the corner of the first data element.

int CXI_Detector::corner_position_valid

Is 1 if corner_position is set and 0 if not.

double CXI_Detector::counts_per_joule

Number of counts recorded per each joule of energy received by the detector.

int CXI_Detector::counts_per_joule_valid

Is 1 if counts_per_joule is set and 0 if not.

CXI_Dataset_Reference* CXI_Detector::data

A reference to a dataset with recorded signal values, or NULL if not set.

CXI_Dataset_Reference* CXI_Detector::data_dark

A reference to a dataset of the image recorded with the shutter closed, used for background subtractions value of the image at each pixel, or NULL if not set.

CXI_Dataset_Reference* CXI_Detector::data_error

A reference to a dataset of the the best estimate of the uncertainty in the data value. Where possible, this should be the standard deviation, which has the same units as the data. Or NULL if not set.

double CXI_Detector::data_sum

Sum of all the elements in the data array. This number is often userful as a cheap measure of data quality.

int CXI_Detector::data_sum_valid

Is 1 if data_sum is set and 0 if not.

name/manufacturer/model/etc. information, or NULL if not set.

double CXI_Detector::distance

Closest distance from the detector to the sample.

int CXI_Detector::distance_valid

Is 1 if distance is set and 0 if not.

CXI_Geometry_Reference* CXI_Detector::geometry

Position and orientation of the center of mass of the detector. This should only be specified for non pixel detectors. For pixel detectors use basis vectors and corner position. Or NULL if not set.

hid_t CXI_Detector::handle

The HDF5 identifier for the detector group.

CXI_Dataset_Reference* CXI_Detector::mask

Not all the pixels in a detector might have the same value. This 32bit mask makes it possible to distinguish different kinds of pixels. The following list defines the meaning of each bit when is it set, as well as the names of constants, defined in cxi.h. Or NULL if not set.
