Introduction

libcxi is a C library intended to help you to read and write CXI files.

1. Installation

Step 1: Obtaining the source

You can obtain the source code of libcxi from GitHub.

Using git you can do:

git clone https://github.com/FilipeMaia/libcxi.git

Alternatively you can directly download the latest verion, in zip format.

Step 2: Building the library

  • Make sure you have HDF5 installed in your machine.
  • Create a build directory inside the source code root directory.
  • Inside the build directory run:
    ccmake ..
  • If all the necessary dependencies are found (HDF5) you can just press c to configure followed by g to generate the Makefiles.
  • Now just run make to actually build the library. If you get any error please post then on the libcxi issues page
  • The library should now be built inside the src directory.

2. Reading CXI files

2.1 Reading a minimal CXI file

libcxi implements a hierarchichal approach to reading CXI files, opening only one group of the HDF5 tree at a time. The following example shows how to read a simple file:

/* Include the necessary header. */
#include <cxi.h>
/* We'll skimp on the error checking for the sake of clarity */
int main(void){
char * filename = "simple.cxi";
/* Open the CXI file for reading */
CXI_File * file = cxi_open_file(filename,"r");
/* Open /entry_1 */
CXI_Entry * entry_1 = cxi_open_entry(file->entries[0]);
/* Open /entry_1/data_1 */
CXI_Data * data_1 = cxi_open_data(entry_1->data[0]);
/* Open /entry_1/data_1/data */
CXI_Dataset * data = cxi_open_dataset(data_1->data);
/* Allocate space to read the data */
float * image = malloc(sizeof(float)*cxi_dataset_length(data));
/* Read the dataset converting to float if necessary */
cxi_read_dataset(data,image,H5T_NATIVE_FLOAT);
/* Sum all the values in the image */
double sum = 0;
for(hsize_t i = 0;i<cxi_dataset_length(data)){
sum += image[i];
}
printf("The sum of all elements of /entry_1/data_1/data is %g\n",sum);
return 0;
}
